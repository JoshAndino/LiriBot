# LiriBot
### Assignment 
In this assignment, I created a LIRI bot. LIRI is a Language Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives you back data.
### Technologies
JavaScript, Node.js,Spotify API, OMDB API, Bands in Town API
### Using Liri
## Spotify node liriTest.js spotify-this-song
<img width="1095" alt="img1" src="https://user-images.githubusercontent.com/35819091/61999443-e6785080-b08e-11e9-88a4-57018302e671.png">

This will search the Spotify API for a song and show the following information about the song in the terminal and to the log.txt file:
node liriTest.js spotify-this-song
* Artist(s)
* The song's name
* A preview link of the song from Spotify
* The album that the song is from

## OMDB node liriTest.js movie-this
<img width="1113" alt="img2" src="https://user-images.githubusercontent.com/35819091/61998765-20dbf080-b083-11e9-9dbd-c848f766f2e1.png">

This will search the OMDB API for a movie name and show the following information in the terminal and to the log.txt file:
  * Title of the movie.
  * Year the movie came out.
  * IMDB Rating of the movie.
  * Country where the movie was produced.
  * Language of the movie.
  * Plot of the movie.
  * Actors in the movie.
